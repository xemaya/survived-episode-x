A pixel art UI element sheet for a Chinese office life game — 16 main menu, pause overlay, and settings UI elements in a 4x4 grid. Style: SFC/16-bit pixel art, visible pixel grid, no anti-aliasing, limited 16-color palette. Mood: dark humor / 喜丧美学. UI motif: 公司内刊 + 打卡机界面, 0-radius right-angle rectangles, 1px borders, copier-printed-form aesthetic. Consistent with O_kpi_review_ui_v01.png and D_character_detail_page_v01.png.

LAYOUT — 4x4 grid on cream/beige background (#E8E0CC). Each cell shows ONE UI element, centered, with 1px border in cubicle gray-blue. Each cell labeled in pixel font below.

THE 16 UI ELEMENTS (top-left to bottom-right):

ROW 1 — Main Menu:
1. 主菜单背景 / Main Menu BG — full-screen mockup miniature: empty pre-dawn office cubicle landscape, fluorescent lights still off, single highlighted-cube workstation, mood = "no one is here yet", title text "活过第 X 集" small in upper-left corner
2. 4 入口按钮纵列 / 4 Menu Buttons Stack — vertical stack of 4 right-angle buttons: "继续上班 / Continue Work", "入职新员工 / New Employee", "查阅人事档案 / Personnel Archive", "下班 / Clock Out", grey-blue with 1px borders, all same width, copier-form style
3. 主菜单标题徽标 / Title Logo Badge — small game logo pixel art: stylized punch-clock with "X" superimposed, chinese subtitle "活过第 X 集" beneath in pixel font
4. 副标题日期/版本 / Version Date Strip — narrow bottom-of-screen strip showing pixel-text "v0.1.0 试用版 / TRIAL", date "2026.05", company logo placeholder

ROW 2 — Pause + Settings BG:
5. "摸鱼中" 暂停叠层 / Pause Overlay — semi-translucent overlay frame with central panel reading "摸鱼中... / SLACKING OFF..." in pixel font, with small pixel art of player slumped in chair, "按 ESC 继续" hint
6. 设置屏背景 / Settings BG — full-screen mockup miniature: clipboard-style settings interface with 4 group panels visible (small), like a corporate forms portal
7. 设置组面板 声音环境 / Settings Group Audio — single panel frame labeled "声音环境 / Audio Environment", contains placeholder for sliders, 1px border with rounded-corner-FREE corners
8. 设置组面板 工作语言 / Settings Group Language — single panel frame labeled "工作语言 / Language", contains placeholder for dropdown, same border style

9. 音量滑块 / Volume Slider — pixel slider control: horizontal track 1px line, square handle (8x8 pixel), tick marks every 25%, "音量: 65%" tiny label

11. 叙事密度切换 / Narrative Density Toggle — three-button segmented control: "完整 / Long" | "速览 / Flash" | "数字 / Numeric Only", currently middle highlighted with double 1px border
12. 按键映射行 / Keymap Row — single row: action label "确认 / Confirm" left, key glyph "[Enter]" middle (rounded-rect key sprite with letter inside), "重置 / Reset" small button right

ROW 4 — Misc UI elements:
13. Tooltip 气泡 / Tooltip Sprite — small pixel-bordered tooltip with arrow point, sample text "档案已满,删除旧条目以新增 / Archive full, delete to add", 1px border
14. 退出确认对话框 / Confirm Quit Dialog — modal box: "下班? 进度已自动保存 / Clock out? Progress saved", two buttons "确认下班 / 取消", same right-angle style as O sheet button
15. 设置组图标四联 / Settings Group Icons (4 icons in this cell) — speaker icon, globe icon, keyboard icon, book icon (each 16x16 pixel art), arranged in 2x2 mini-grid within the cell
16. 重置默认按钮 / Reset to Defaults Button — distinctive button "恢复默认 / Reset to Defaults", red-tinted text on grey-blue base, 1px border, slightly more cautionary appearance

PALETTE: 打工人黄 #C8A85A, 灰蓝 #5A7080 (UI dominant), 棕 #7A5838 (wood/paper trim), 白 #E8E0CC (background, labels), 屏幕蓝 #2C4A6E (main menu BG tint, slider track), 老板金 #E0B050 (max 3% — title logo accent), 警告红 #C83428 (reset button text + warning).

VISUAL JOKES (1-3 px subtle):
- Main menu BG single highlighted workstation = the player's permanent station, even when they're not there
- Pause overlay: player sprite has tiny phone reflection on face (he is indeed slacking)
- Volume slider sample at 65% (specific number = bureaucratic precision)
- Confirm quit dialog: "下班" sounds final because in this game it kind of is
- Reset button red-tint warns it's destructive (rare red usage outside KPI/GO)
- Title logo's "X" suggests both "episode X" and "X marking the spot of failure"

CRITICAL CONSTRAINTS:
- All right-angle 0-radius rectangles per UI motif
- 1px borders, never gradient or shadow-heavy
- Buttons use stamp-effect double-border on hover (carry over from O sheet)
- Pause overlay must be diegetic-flavored, not a generic "PAUSED" banner
- Each cell visually framed, label below

TOP TITLE BANNER: "主菜单 · 暂停 · 设置 / Main Menu · Pause · Settings UI" pixel font, centered, dark cubicle blue text on cream.
