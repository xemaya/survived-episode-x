A pixel art NPC event CG portrait sheet for a Chinese office life game — 8 NPCs × 2 emotional states = 16 medium-shot portrait CGs in a 4x4 grid. Style: SFC/16-bit pixel art with slightly more detail than the small-icon NPC atlases (these are larger event-CG portraits), still SFC/16-bit aesthetic. Mood: dark humor / 喜丧美学. Consistent with character_sheet_player_v01.png style — these are the "cinematic" event portraits used during long dialog events, larger than the diegetic HUD faces. Each portrait shows shoulders + head with more facial expression detail.

LAYOUT — 4x4 grid on cream/beige background (#E8E0CC). Each cell shows ONE NPC at ONE emotional state, framed in a 1px portrait border (mimicking corporate ID photo aesthetic). Each cell labeled with Chinese NPC name + emotion + English in pixel font below.
