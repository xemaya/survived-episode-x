A pixel art NPC lifecycle and animation atlas for a Chinese office life game — 16 NPC lifecycle and animation frames in a 4x4 grid. Style: SFC/16-bit pixel art, visible pixel grid, no anti-aliasing, limited 16-color palette. Mood: dark humor / 喜丧美学. Consistent with reference sheets A_npc_archetypes_v01.png, L_npc_faces_set1_v01.png, M_npc_faces_set2_v01.png — same characters at same scale.

LAYOUT — 4x4 grid on cream/beige background (#E8E0CC). Each cell shows ONE NPC frame (mostly LEAVING_ANNOUNCED state with cardboard box, plus a few animation frames). Each cell labeled with Chinese + English in pixel font below.
