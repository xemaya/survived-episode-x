A pixel art HUD diegetic prop sheet for a Chinese office life game — 16 calendar/attendance/notice elements arranged in a 4x4 grid. Style: SFC/16-bit pixel art, visible pixel grid, no anti-aliasing on outlines, limited 16-color palette. Mood: dark humor / 喜丧美学 (funeral-as-festival). Consistent with prior reference sheets like A_npc_archetypes_v01.png and J_hud_workstation_props_v01.png.

LAYOUT — 4x4 grid of individual prop sprites on a clean cream/beige background (#E8E0CC). Each cell shows ONE prop, centered, with subtle 1px drop shadow. Each cell labeled with Chinese + English in pixel font below the prop.

THE 16 PROPS (top-left to bottom-right):

ROW 1 — 墙上考勤板 (Attendance Board) sub-mode states 1-4:
1. 平日上班 / Day Normal — green LED dot in upper-left, "在岗 / ON" pixel text, plain grid of name slots filled with green checkmarks
2. 工作繁忙 / Day Busy — yellow LED, several slots show "外出/OUT" red label, one slot blank, slight "tilt" of one row
3. 加班 / Overtime — red LED, time stamp "21:47", multiple slots show overtime hour numbers (3.5h / 4.0h)
4. 周末 / Weekend — gray LED, board mostly empty, one or two diehard slots still marked "在/IN"

ROW 2 — 墙上考勤板 sub-mode states 5-8:
5. 月末考核 / KPI Review — purple LED, top label "月度考核中", names with 1-3 stars next to them, no checkmark column
6. 调休日 / Day Off — pale blue LED, "休/REST" text in most slots, dust on board edges (1px specks)
7. 解雇 / Game Over — red LED steady, single slot crossed out with red X, name still legible
8. 主菜单待机 / Main Menu Idle — no LED lit, board completely blank, faint coffee stain in corner

ROW 3 — 桌面日历 (Desk Calendar) frames:
9. 月初第一天 / Month Day 1 — small wooden flip calendar showing red "1", current week highlighted, today circled in pencil
10. 周中工作日 / Mid-Week — page showing "15", thumbtack pinned to one date with "考核 / Review" in tiny pencil
11. 月末倒数 / Month End — page showing "28", red marker circle around it, page corner curled
12. 周末标记 / Weekend Marked — calendar page showing weekend cells faintly grayed, current Saturday with subtle highlight

ROW 4 — 通告板 + 横幅 (Notice Board + Banner):
13. 周年俗艳粉横幅 / Anniversary Banner — narrow horizontal banner in tacky pink (#E0A0A0) with gold trim, "司庆 X 周年" pixel text, paper streamers hanging
14. 通告板 全新 / Notice Board Fresh — corkboard with crisp white memo "通告 / NOTICE" pinned center, thumbtacks bright red
15. 通告板 微旧 / Notice Board Faded — same corkboard, memo paper yellowed at edges, one corner curling
16. 通告板 陈年 / Notice Board Ancient — memo nearly fully sun-bleached, corners torn, thumbtacks rusted brown, one fly on board (1px dot)

PALETTE: 打工人黄 #C8A85A, 灰蓝 #5A7080, 棕 #7A5838 (board frames), 白 #E8E0CC (paper), 屏幕蓝 #2C4A6E, 老板金 #E0B050 (LED highlight only, max 3% pixel coverage), 警告红 #C83428 (LED + red X), 周年粉 #E0A0A0 (banner only).

VISUAL JOKES (1-3 px subtle):
- The "Day Off" board has dust because no one cleans it
- Anniversary banner streamers are slightly wilted (1px droop)
- "Game Over" attendance slot still shows the person's name — they were here
- "Notice Board Ancient" has a fly because it's been there long enough to attract one
- Calendar "Month End" pencil circle is shakily drawn (3 small offsets)

CRITICAL CONSTRAINTS:
- All 8 attendance board states must be silhouette/color-distinct at thumbnail
- The 4 notice board fade levels show progression of bureaucratic forgetting
- Anniversary banner is the ONE place gaudy pink is allowed (Pillar 4 irony anchor)
- NO triumphant styling — these are bureaucratic objects that count time passing
- Each cell visually framed but borderless (just the prop on cream, label below)

TOP TITLE BANNER: "考勤 · 日历 · 通告 / Attendance · Calendar · Notice" pixel font, centered, dark cubicle blue text on cream background.

Output: PNG, 1024x1024, single composite 4x4 sprite sheet, ready to be cropped into individual diegetic HUD sprites.
