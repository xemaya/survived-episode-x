A 3x3 grid character archetype reference sheet for a pixel art Chinese office life game. Style is SFC/16-bit pixel art, visible pixel grid, no anti-aliasing on outlines, limited 16-color palette.

LAYOUT: 3 rows x 3 columns, each cell shows ONE distinct office NPC archetype, half-body (waist up). Cream/white background, thin pixel borders between cells, professional reference sheet style. Each cell has a Chinese label (primary) + English label below.

The 9 archetypes (read across rows top to bottom):
