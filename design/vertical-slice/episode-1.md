## 切片设计目标

1. **建立笑天的"声音"**——名字段子 / 绿萝 / 富士山头像 / "周一上午 9:14 地球继续转动"——所有这些"声音锚"在 E1 集中出现一次，让玩家学会盲读辨认笑天的内心独白
2. **8 个 NPC 第一次出场**——Lisa / David / 王总监 / 李阿姨 / Vivian / IT 小马 / 老周 / 妈妈 各自完成 Archetype A 的 specific micro-detail anchor（Zoe 留到 E2，林姐 S1 不出场）
3. **9 : 1 笑泪比**——几乎全笑。唯一的轻扎留给周日笑天独自去公司浇绿萝那一刻
4. **Fake-out 周三晨会**——王总监今天没讲 KPI。玩家以为大事要发生，结果是日常。这是这游戏 surface 与 expectation 错位的笑点教学
5. **不要发生的事**——Lisa 没有任何"靠近笑天"以外的状态变化（不剪短发 / 不被 cue / 不透露 HR）；David 没有抢功（那是 E2）；王总监没有讲反向 KPI 真相（那是 E2 周三晨会）
