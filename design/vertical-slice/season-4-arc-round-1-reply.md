## 3. Q3 答 — Lisa 微信 "4 句模式" 频率 OK

**Decision**: ✅ keep — 路径 A 每集 1 次, 4 句以内。

理由:
- npcs.md §1 "S5+ 偶尔回办公室办手续" — "偶尔" 跟 "每集 1 次" 跨 4 集 = 跨 1 个月 4 次, 是 weekly cadence (合理 spillover)
- "4 句以内" = npcs.md §1 "她对笑天有 boundary, 不让 over-involve" 的 verbal expression
- 整 S4 16 句话总量 — 比 S1-S3 整季 Lisa 出场量 (E5-E12 大约 200+ 行) 大幅减少 — reflects "她已经 move on" properly

**注意**: 路径 A Lisa 微信不要写 "笑天我想你了 / 跟林姐不熟悉" 类 emotional line — 那是反向 movement。她在新部门要建立新关系, 微信 4 句应保持 "transactional friendly" tone:
- 笑天问候 "新部门第一天还行吗"
- Lisa 回 "还行 + 谢谢笑天"
- 笑天回 "好"
- Lisa 收尾 "嗯 + (新部门 specific 细节)"

W6 outline 的 Lisa 4 句模板已 fit 这个 spirit — 不变。

## 4. Q4 答 — Path E silence < 5 lines 整季 OK

**Decision**: ✅ keep silence — Pillar 3 极致 spirit。

理由:
- 路径 E 玩家 S2-S3 已 mute Lisa (lisa_score < -5), 她也 mute 笑天

- 玩家路径 E 的 emotional reward = "你成了'她不存在'的版本" — 这是 Pillar 3 的具象化, 不 dilute

**给 S4 ink writer 的提示**: E13 Vivian "嗨～她上周走了" 后, 路径 E 玩家整季 Lisa 0 mention。**不要为了"叙事完整"加 silence-breaking line** — 玩家选择路径 E 就 deserve silence。

## 5. Q5 答 — 上坟 mention OK (有 constraint)

**Decision**: ✅ keep "上坟" mention, **加 strict constraint** 给 S4 ink writer。

理由:
- "上坟" 是清明节 standard cultural verb, 妈妈视频 mention 自然
- 跟 npcs.md §9 character note "不在剧本里 expose 爸爸 8 年前去世" 不冲突 — 因为 mention 的是**清明流程行为**, 不是 emotional anchor (爸爸已逝)
- 妈妈把 "你姨他们要去" 替换 "你爸" 直接 mention — 是 indirect deflection, character-consistent

**S4 ink writer 必守 constraint** (写 episode-13.ink E13 周日 mom 视频 stitch 时):

1. 妈妈 dialog: "**你姨他们要去你爸坟上 — 你今年回不回来**" 是合规 line。**不要**改成 "你爸坟上你回去看看", 那个 directly 让妈妈说出 "你爸" 作为 actor + 给 emotional anchor
2. 笑天 internal monologue 反应 = 选项 3 选 1 后的 "下班" 自然 transit。**不要**插入 italic monologue 类 "_我又一次没回去_" 或 "_8 年前妈妈第一次..._"
3. 笑天选项后 effect: KPI / state +/- 数值 ok, **不要**写 hidden flag like `dad_grave_visit_decline += 1` (会 enable 未来 episode 用 "爸爸已逝" 作为 narrative payoff — 违反 character note)
4. 后续 episode (S5+) 不要 use "上坟" 作为 callback motif — S4 E13 是 one-time mention, 不形成 spiral

W6 outline 当前写法 (E13 周日 + E14 妈妈替亲戚解释) 完全 fit 这个 constraint。**保留**, 不 change。

## 6. Q6 答 — 林姐 E15 茶水间偶遇 0 句话 OK

**Decision**: ✅ keep 0 句话 — deliberate restraint > expand。

理由:
- npcs.md §10 锁了 "林姐对笑天 0 主动 dialog" 跨整 series
- S3 finale (路径 A) + S4 E15 = **2 次 visual 接触, 0 句话** — 跨 season callback consistency 完美

W6 outline 的 E15 林姐茶水间偶遇 stitch 写得对: 点头 + 0.3 秒看笑天工位方向 + 拿水离开。**保留**, 不 expand。
