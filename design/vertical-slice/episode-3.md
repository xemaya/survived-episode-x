## 切片设计目标

1. **NPC C Vulnerability 集中爆发集**——9 个 NPC（除林姐）每人在 E3 各有一次"露馅"。但**没有一个是直接对玩家露**——全部是被偷看 / 偷听 / 意外撞见。这是 C archetype 的核心规则
2. **老周"过完今天"**——E3 周四笑天主动找老周对话——老周说 1 句话："过完今天。"**这是 S1 全季唯一一次笑天和老周的对话**（per `season-1-arc.md` §3.8 + §10 设计自检"老周'过完今天'那句话**全 S1 只出现 1 次**——多了变 mentor")
3. **Lisa 剪短发**（兑现 E2 cliffhanger）——周一她出现，剪短了，"想换个心情"。**心理学梗：人在重大决定前会先剪头发**
4. **6:4 笑泪比**——笑减少。HR 暗线 + 王总监工位灯还亮 + 李阿姨"这周又走了仨" + 妈妈视频上海买房 = 4 个扎点
5. **不要发生的事**：HR 找 Lisa 谈话**绝不能出现**（这是 S2 finale = E8 的 beat）；Lisa 走/留**绝不能出现**（这是 S3 finale = E12）
6. **Cliffhanger**：下周一就是月末 KPI Review。**E4 finale 的 5 路径揭晓**
