# 垂直切片 — 口吻规则 (Tone Bible)

> Status: 第 2 版（去禁词化）
> Author: Game Designer (Claude)
> Last Updated: 2026-05-04
> Source: design/gdd/game-concept.md Pillar 4 + npc-relationship-system.md (NPC 算计原则) + protagonist.md（主角声音）
>
> **写在最前**：第 1 版 11 条禁词 + `_IRONY` / `_BUREAUCRATIC` 后缀系统全部撤销——规则越机械越绕不过去。这一版只定 **5 条原则**。文案审校时不查词典，**靠人脑判断 5 条原则**。
>
> **笑/泪比例**：8 : 2。surface 是荒诞观察 + 主角自嘲，悲剧 anchor 留给关键弧光（如 Lisa 月末）。每天 ≥ 2 个真笑点 / ≤ 1 个扎点是基线。

---

## 1. 五条原则

### 原则 1：主角是观察者，不是英雄

主角"撑过去"，不"做到了"。
主角"麻木了"，不"成长了"。
主角"还在"，不"突破了"。

> 这游戏没有禁词，只有**禁基调**。"努力 / 成长 / 突破 / 完美 / 战略 / 友谊 / 共赢"这些词频繁出现在主角内心独白或事件叙述里 = **主角变英雄** = 失守。
>
> 但这些词出现在 **王总监 PUA 话术 / Zoe HR-speak / 公司公文体 / David 自我感动**里 = **正确**——这就是这个游戏要讽刺的语料本身。
>
> 区分的方式不是查词，是判断**这话从谁的嘴里出来**。从主角嘴里 / 内心独白里出来 = 失守；从这个荒诞世界的角色嘴里出来 = 通过。

**陈笑天的口吻关键词**：观察者位置 + 自嘲优先 + 不煽情 + 反高潮。详见 `protagonist.md` §4。

---

### 原则 2：NPC 是为自己活的

每条 NPC 对白审校时问一个问题：

> **她说这句话是为了她自己，还是为了玩家？**

- 为她自己 = 通过
- 为玩家（鼓励 / 帮助 / 利他 / 关怀） = 改写

NPC 的"善意"如果出现，必须**带利己动机**：
- Lisa 给笑天奶茶 = 因为她想跟你建立关系（她孤独）
- David 帮笑天 = 因为他想用你（信息 / 借力 / 试探）
- 王总监关心 = 在收集情报
- Zoe 提醒 = 在执行流程
- 李阿姨善意 = 她在跟自己的回忆说话，你听见了

NPC **永远不是工具人**。详见 `npcs.md` 各人的"内在动机"。

---

### 原则 3：主语翻转 — 数值变化用 NPC / 物 / 时间陈述

**所有数值变化的文案必须以 NPC / 物 / 时间为主语，不是"你"或"系统"**。

❌ 失守：
> 你和 Lisa 的关系 +5。

✅ 通过：
> Lisa 转过头看了你一眼，没说话。

具体规则：
- **关系变化**：写 NPC 行为（"她笑了一下" / "他没回头" / "她又坐回工位"）
- **KPI 变化**：在事件正文里**完全不提**——只在每日日报里以"今日 +N"出现
- **AP 变化**：必要时写在系统标注（`AP -1`），事件正文不提
- **环境变化**：用"工位空了" / "桌面擦干净了" / "茶水间今天没人"，不用 UI 标语

---

### 原则 4：写真不写好 — 最锋利的话是最简单的真实

形容词越少，刺越深。

❌ "她绽放了温暖的微笑。"
✅ "她笑了一下，没说话。"

❌ "她的离去让办公室充满了沉重的氛围。"
✅ "工位空了一周。"

❌ "李阿姨眼中流露出深深的悲悯，仿佛看穿了所有人的命运。"
✅ "上一个坐这位置的也是这么想的。"

**HR-speak / PUA 话术 / 周报体保留得越接近现实越好笑——不要"加效果"**。让冰冷的官腔自己来杀人：
- 真实的 HR："这周方便的话签个字。" — 直接抄
- 真实的王总监："我相信大家的潜力。" — 直接抄
- 真实的周报："完成多项基础性工作。" — 直接抄

这些话在现实里就在杀人。你写真就够了。

---

### 原则 5：朋友圈测试 — 真正的中文打工人会截图发朋友圈吗？

每段文案写完后，问自己：

> **真正的中文打工人，会不会把这段截图发朋友圈？**

- 会 = 通过
- 不会 = 重写

这是最锐利的检验。它同时检验**笑/泪 8:2 比例**：

- 发"哈哈哈这就是我办公室"的截图 = 笑点（占 8）
- 发"扎心了"的截图 = 泪点（占 2）
- 都不发 = 既不笑也不扎 = 文字浪费 = 删

**月末读完所有文案，回头数一遍：截图清单里"哈哈哈"型 vs "扎心"型，比例是不是 8:2？**

## 2. 玩家叙事的人称规则

### 必用：第二人称 "你"

> 你看了一眼 Lisa 的工位。
> 你打卡的时候发现机器响了一声。
> 你想起来周一就是新月份。

### 禁用：第一人称 "我"

❌ "我看了一眼 Lisa"——破代入
❌ "我心里一紧"——煽情

### 禁用：第三人称（除了在描述 NPC 时）

❌ "玩家做了选择"——破第四面墙
❌ "主角不知道"——把玩家当工具

### 心理活动写法

需要写玩家"心里想什么"时，用**斜体**（rendered as italic）或破折号引出，**不让"你"做主语**：

✅ `_后来你想起来：人在重大决定前会先剪头发。_`
✅ `—— 你不知道说的是谁。`
✅ `—— 但你听见了。`

❌ `你心想："她可能要走了。"`（直接写心理活动 = 太重）

## 3. 选项写法规则

### 选项必须**短**且**带行动语气**

**长度 target：≤ 6 字**（v2.1 update — 从 v2 的"≤ 4 字 strict"放宽）。
**专用 phrase 例外**：中文职场招牌梗 / 英文混用 phrase（"Alt+Tab 装打字" / "cc 王总监" / "已收到 + 闹钟提前"）可以超 6 字，**前提是损失梗 = 损失场感**。审校时问："这个 phrase 短一字会不会变 generic？" 会 → 保留长版本。

✅ `[接过来]`（3 字）
✅ `[我也忙]`（3 字）
✅ `[假装没听见]`（5 字）
✅ `[Alt+Tab 装打字]`（专用例外 — 7 字但梗值满）
✅ `[cc 王总监]`（专用例外 — 5 字英文混用）
✅ `[已收到 + 闹钟提前]`（专用例外 — 9 字但完整表达"表演 alive 升级"）

❌ `[你帮 Lisa 看 PPT，扣 1 AP]`（解释成本，应该藏在结果里）
❌ `[选择善良的回应]`（评价性，违反 anti-pillar 4）
❌ `[展现你的能力]`（励志，违反 anti-pillar 4）

**v2.1 改动 motivation**：Round 2 review 发现 daily-choices.ink 有 ~10 个 stitch 选项 6-9 字（如 "bullshit bingo" / "删邮件 删快递条 删一切"），全部是 高梗值职场流行语。如果 strict 4 字 truncation 会变成 "bingo" / "删一切"——梗损失。Designer accept 这种例外，但要 case-by-case 判断 "短一字是否变 generic"。

### 选项之间必须**意义不同**

3 个选项不能是"同意 / 同意但语气不同 / 还是同意"——必须是真选择。

模板：
- A：积极介入（高代价，可能正面后果）
- B：保守应对（中代价，中性后果）
- C：彻底回避（低代价，可能负面后果）

但**不要每次都套这个模板**——会无聊。混入：
- 有时只 2 个选项（强迫站队）
- 有时所有选项都"扎"（道德困境）
- 有时根本没选项（**Pillar 2 极致**——没有选项也是设计）

### 选项后果文案要**克制**

✅ 选 [我也忙] 后：`David："好吧，我自己搞。"`
❌ 选 [我也忙] 后：`David 失望地走了。你失去了一个朋友的信任。`

后果用 NPC 行为 1-2 行陈述，不评价、不解释、不煽情。**让玩家自己脑补**。

## 4. 节奏档位（叙事密度）

GDD 定义了三档：`flash` / `standard` / `verbose` / `numeric_only`。在垂直切片中：

### Flash（闪现，~2-3 行）
**用于**：日常 flavor、过渡、不影响数值的瞬间
**示例**：王总监 Day 1 那个"加油啊"

### Standard（标准，~5-8 行）
**用于**：有选择的事件、关系变化、剧情推进
**示例**：Lisa 茶水间吐槽、David 抢功

### Verbose（长，~10+ 行）
**用于**：剧情高潮、月末关键、说出系统真相
**示例**：Day 3 王总监"潜力是你今天的最好成绩……"那段

### Numeric_only
**用于**：玩家选择"我不要剧情"模式时所有事件压缩为数字
**示例**：`Lisa +3` 而不是"Lisa 抬头看了你一眼"

垂直切片不实装 numeric_only，等到 P5 玩家有"调节叙事密度"设置时再做。但**所有事件文案要写好以便未来能压缩**——意味着关键数值变化必须能从文案抽离。

## 5. 情绪曲线规则

每个游戏内的"周"应该有曲线：

```
周一 → 周三：设定 + 锚点（认识 NPC、第一次冲突）
周三晨会：高峰 1（王总监讲反向 KPI 真相）
周四 → 周五：危机酝酿（Lisa 头发剪短、HR 暗线、David 暴露）
周五傍晚：高峰 2（Lisa 透露 HR）
周末：缓冲 + 抉择（陪不陪 Lisa）
周日（月末）：高峰 3（结局）
```

**避免**：每天都是高峰（玩家麻木）、整周都是缓冲（玩家无聊）

**追求**：缓-急-缓-急-急的节奏

## 6. 笑/泪 8:2 配比清单（Day 1-7 切片）

笑点 ≥ 14 / 扎点 ≤ 4 = 8:2 比例。

### 笑点（截图发"哈哈哈这就是我办公室"）

| Day | 时刻 | 笑点 |
|---|---|---|
| Day 1 | 王总监叫不准"陈笑天" | "小笑啊…陈天啊…差不多差不多" |
| Day 1 | 自我介绍尴尬 | 别人愣 0.5 秒后"哦哦挺好挺好" |
| Day 1 | 工位前任绿萝 | 笑天每天浇 3 滴水（"它走了我还在") |
| Day 1 | 茶水间咖啡机 IT 工单 | "已派单"3 周不动 |
| Day 1 | 王总监"加油啊" | 老板把记住下属名字当 KPI |
| Day 2 | David 群里假感谢 | "@所有人 感谢 David 加班" + 笑天内心"那是我改的" |
| Day 2 | David "5 分钟的事" | 永远不止 5 分钟 |
| Day 2 | David 自我感动 | "我这种人就是闲不住" |

| Day 3 | 王总监 PPT | "团队 BLUEPRINT" 第一页只有一个箭头 |
| Day 3 | 反向 KPI 真相 | "潜力 = 你今天的最好成绩是明天的最低标准" 黑色幽默 |
| Day 3 | David 抢功 0.5 秒举手 | 速度比 Wi-Fi 快 |
| Day 4 | Zoe 工位反差 | "工作快乐" 标语 + 偷刷小红书"裸辞攻略" |
| Day 4 | David 厕所偶遇 | 俩人尴尬互相点头 |
| Day 4 | David 唯一露馅 | 试探"潜力一般"被反问后秒岔"你保温杯哪买的" |
| Day 5 | 福利水果盘 | 草莓周（融资到位）vs 苹果周 |
| Day 5 | David 周五写下周计划 | 4 点已经在写"周一计划" |
| Day 6 | 笑天周末 11 点起床 | "世界上有人在游艇上，我在补觉。公平。" |
| Day 6 | 妈妈周日视频固定剧本 | "吃了吗？" "吃了。"（他没吃）|
| Day 7 | 王总监又叫错名字 | 月末 running gag 收尾 |
| Day 7 | 笑天想象自己被裁 | "至少我会比 Lisa 哭得难看。" |

总：~21 个笑点位置（部分按选项触发）

### 扎点（截图发"扎心了"）

| Day | 时刻 | 扎法 |
|---|---|---|
| Day 4 | Lisa 剪短发 | 心理学梗 setup（重大决定前剪头发）|
| Day 5 傍晚 | Lisa 透露 HR 找她 | 主线扎心 |
| Day 5 晚 | 李阿姨"这周又走了仨" | Pillar 3 入口，不刻意 |
| Day 7 月末 | Lisa 工位空 + 李阿姨多拖一遍 | 安静的悼念 |

总：4 个扎点。Lisa 弧光是核心，**不动**。

### 比例自检
- 21 笑 : 4 扎 ≈ **5.25 : 1**（接近 8:2 = 4:1，略偏笑）
- 通过——略偏笑比偏泪好（用户明确指示）

### 朋友圈测试
- 愿意截图发"哈哈哈"的笑点 ≥ 8 个 = 通过
- 愿意截图发"扎心了"的扎点 ≥ 2 个 = 通过

## 8. Tone Bible 自检清单（每段文案写完都过一遍）

5 条原则 + 4 条工艺细节，**任意一条不通过 → 重写**。

**5 条原则**：

- [ ] **原则 1（主角是观察者，不是英雄）**：主角的内心独白 / 事件叙述里有没有"励志/突破/成长/完美"基调？这些词只能从 NPC 嘴里讽刺地出现
- [ ] **原则 2（NPC 是为自己活的）**：NPC 这句话是为她自己，还是为玩家？为玩家 = 改写
- [ ] **原则 3（主语翻转）**：数值变化的文案主语是 NPC / 物 / 时间，不是"你"或"系统"
- [ ] **原则 4（写真不写好）**：形容词够少吗？HR-speak / PUA 话术有没有"加效果"，能不能更接近现实？
- [ ] **原则 5（朋友圈测试）**：真正的中文打工人会截图发朋友圈吗？

**4 条工艺细节**：

- [ ] **第二人称叙事**：玩家叙述用"你"，不是"我"或"主角"
- [ ] **选项克制**：选项 ≤ 6 字 target（专用职场梗 phrase 可例外，per §3）+ 不评价 + 不解释
- [ ] **后果克制**：后果 1-2 行陈述，让玩家自己脑补
- [ ] **笑/泪比例**：这一段如果是文本主体，偏 笑（surface 观察 + 主角自嘲）还是偏 扎（高潮 anchor）？整月加起来要 8:2

## 写在最后

这游戏的写作不像写小说。**最锋利的话往往最简单**：

> "她笑了一下，没说话。"
> "工位空了一周。"
> "上一个坐这位置的也是这么想的。"

不要试图"写得好"。**写得真**就够了。这才是反 KPI 的精髓——**做得好不如做得真，做得真不如刚刚好**。
