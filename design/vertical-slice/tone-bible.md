## 1. 反主角口吻 — 红线 1 落地规则

### 禁词清单（看到 = 必改）

| 禁词 | 为什么 | 替代 |
|---|---|---|
| 你做到了 | 励志主角范 | "事情完成了" / "工作做完了" |
| 完美 | 假赞美 | "差不多" / "凑合" / "及格" |
| 战略 | bullshit bingo | "做法" / "方案" |
| 友谊 | 主角中心叙事 | （不写。改成 NPC 行为：她坐过来 / 她递过来）|
| 成长 | 反 anti-pillar 1 | "适应" / "习惯" / "麻木" |
| 突破 | 同上 | "撑过" / "还在" |
| 加油 / 努力 | 励志 | （只能由王总监 PUA 时说，加 _IRONY 标记）|
| 一定 / 肯定 | 假承诺 | "应该" / "可能" |
| 团队 / 大家 | 集体主义 PUA | （只能由王总监 / Zoe 说，加 _IRONY）|
| 共赢 / 双赢 | bullshit | （同上）|
| 命运共同体 | 同上 | （同上）|

### `_IRONY` 后缀规则

某些禁词在特定上下文（王总监 PUA / Zoe HR-speak）里**必须**出现——这是这个游戏要讽刺的语料本身。这种使用要在文案 corpus 里标注 `_IRONY` 后缀，绕过 lint。

例：
- `王总监.开会_IRONY = "我们是命运共同体"`
- `王总监.路过工位_IRONY = "潜力一般"`
- `Zoe.HR_面谈_BUREAUCRATIC = "已为您协调岗位适配方案"`

读者（玩家）能理解这些话**就是**讽刺；如果出现在玩家的内心独白或描述性叙述里，就是失守。

### `_BUREAUCRATIC` 后缀规则

HR-speak / 公司公文体专属。例：
- `已为您调整岗位适配度`
- `这边的流程是这样的`
- `理解一下`
- `走完流程`
- `这周方便的话签个字`

听起来礼貌，但读出来全是冰冷的执行——这就是这个游戏的反派之一。

## 2. 主语翻转规则

**核心**：所有数值变化的文案必须以 **NPC / 物 / 时间** 为主语，不是以"你"或"系统"为主语。

### ❌ 失守示例

```
你和 Lisa 的关系 +5。
```

### ✅ 通过示例

```
Lisa 转过头看了你一眼，没说话。
```

```
工位上的玩偶位置和昨天不一样。
```

```
打卡机响了一声。
```

### 详细规则

- **关系变化**：永远写 NPC 的行为（"她笑了一下" / "他没回头" / "她又坐回工位"），从不写"+N 关系"
- **KPI 变化**：永远写"今天 + 多少分"作为日报数据，事件中**完全不提**
- **AP 变化**：必要时写"AP -1"作为系统标注，但事件正文不提
- **环境变化**：用"工位空了" / "桌面擦干净了" / "茶水间今天没人" 等，**不**用 UI 标语

## 3. NPC 算计原则（GDD 已明确）

每条 NPC 对白审校时问一个问题：

> **她说这句话是为了她自己，还是为了玩家？**

### ✅ 通过（NPC 在算计）

- Lisa: "我猎头那边明天面试，你别和老板讲啊"（用你做信息屏障）
- David: "兄弟，帮我看下这 PPT 行不？"（在剥削你）
- 王总监: "今天月末了，你的 KPI 怎么样？"（在收集情报）
- Zoe: "你这边方便的话……"（在执行流程）
- 李阿姨: "上一个坐这位置的也是这么想的"（她在跟自己的回忆说话，你听见了）

### ❌ 失守（NPC 在帮玩家）

- Lisa: "我相信你能搞定下季度！" → 励志，违反
- David: "我觉得你比我有潜力，你应该出头" → 利他，违反
- 王总监: "小心 KPI，别太累" → 关怀，违反（他不 care）
- Zoe: "我私下提醒你一下" → HR 不会主动泄密，违反
- 李阿姨: "年轻人加油" → 鸡汤，违反

## 4. 玩家叙事的人称规则

### 必用：第二人称 "你"

> 你看了一眼 Lisa 的工位。
> 你打卡的时候发现机器响了一声。
> 你想起来周一就是新月份。

### 禁用：第一人称 "我"

❌ "我看了一眼 Lisa"——破代入
❌ "我心里一紧"——煽情

### 禁用：第三人称（除了在描述 NPC 时）

❌ "玩家做了选择"——破第四面墙
❌ "主角不知道"——把玩家当工具

### 心理活动写法

需要写玩家"心里想什么"时，用**斜体**（rendered as italic）或破折号引出，**不让"你"做主语**：

✅ `_后来你想起来：人在重大决定前会先剪头发。_`
✅ `—— 你不知道说的是谁。`
✅ `—— 但你听见了。`

❌ `你心想："她可能要走了。"`（直接写心理活动 = 太重）

## 5. 选项写法规则

### 选项必须**短**且**带行动语气**

✅ `[接过来]`
✅ `[我也忙]`
✅ `[假装没听见]`

❌ `[你帮 Lisa 看 PPT，扣 1 AP]`（解释成本，应该藏在结果里）
❌ `[选择善良的回应]`（评价性，违反 anti-pillar 4）
❌ `[展现你的能力]`（励志，违反 anti-pillar 4）

### 选项之间必须**意义不同**

3 个选项不能是"同意 / 同意但语气不同 / 还是同意"——必须是真选择。

模板：
- A：积极介入（高代价，可能正面后果）
- B：保守应对（中代价，中性后果）
- C：彻底回避（低代价，可能负面后果）

但**不要每次都套这个模板**——会无聊。混入：
- 有时只 2 个选项（强迫站队）
- 有时所有选项都"扎"（道德困境）
- 有时根本没选项（**Pillar 2 极致**——没有选项也是设计）

### 选项后果文案要**克制**

✅ 选 [我也忙] 后：`David："好吧，我自己搞。"`
❌ 选 [我也忙] 后：`David 失望地走了。你失去了一个朋友的信任。`

后果用 NPC 行为 1-2 行陈述，不评价、不解释、不煽情。**让玩家自己脑补**。

## 6. 节奏档位（叙事密度）

GDD 定义了三档：`flash` / `standard` / `verbose` / `numeric_only`。在垂直切片中：

### Flash（闪现，~2-3 行）
**用于**：日常 flavor、过渡、不影响数值的瞬间
**示例**：王总监 Day 1 那个"加油啊"

### Standard（标准，~5-8 行）
**用于**：有选择的事件、关系变化、剧情推进
**示例**：Lisa 茶水间吐槽、David 抢功

### Verbose（长，~10+ 行）
**用于**：剧情高潮、月末关键、说出系统真相
**示例**：Day 3 王总监"潜力是你今天的最好成绩……"那段

### Numeric_only
**用于**：玩家选择"我不要剧情"模式时所有事件压缩为数字
**示例**：`Lisa +3` 而不是"Lisa 抬头看了你一眼"

垂直切片不实装 numeric_only，等到 P5 玩家有"调节叙事密度"设置时再做。但**所有事件文案要写好以便未来能压缩**——意味着关键数值变化必须能从文案抽离。

## 7. 情绪曲线规则

每个游戏内的"周"应该有曲线：

```
周一 → 周三：设定 + 锚点（认识 NPC、第一次冲突）
周三晨会：高峰 1（王总监讲反向 KPI 真相）
周四 → 周五：危机酝酿（Lisa 头发剪短、HR 暗线、David 暴露）
周五傍晚：高峰 2（Lisa 透露 HR）
周末：缓冲 + 抉择（陪不陪 Lisa）
周日（月末）：高峰 3（结局）
```

**避免**：每天都是高峰（玩家麻木）、整周都是缓冲（玩家无聊）

**追求**：缓-急-缓-急-急的节奏

## 8. "会心一笑"清单（Day 1-7 切片至少 5 个）

| Day | 时刻 | 笑点 |
|---|---|---|
| Day 1 | 王总监"加油啊" | 老板把记住下属名字当 KPI |
| Day 2 | David "5 分钟的事" | 永远不止 5 分钟，且最后他抢功 |
| Day 3 晨会 | "潜力 = 今天的最好成绩是明天的最低标准" | 把反向 KPI 直接说出来，但用 PUA 包装 |
| Day 4 | Lisa 剪短发 | 心理学梗（重大决定前剪头发）|
| Day 4 中午 | Zoe 工位"周五签字" | setup—在 Day 5 才 payoff |
| Day 5 | David 试探 "你被说过 '潜力一般' 吗" | 卷王也是受害者 |
| Day 5 晚 | 李阿姨"这周又走了仨" | Pillar 3 入口，不刻意 |
| Day 7 | 李阿姨多拖一遍 Lisa 工位 | 安静的悼念 |

**朋友圈测试**：哪几条你愿意截图发朋友圈？**愿意 ≥ 3 条**——切片 tone 通过。

## 10. Tone Bible 自检清单（每段文案写完都过一遍）

- [ ] **朋友圈测试**：你愿意把这段截图发朋友圈吗？
- [ ] **NPC 算计原则**：NPC 的话是为她自己，还是为玩家？
- [ ] **主语翻转**：数值变化的文案主语是 NPC / 物 / 时间，不是"你"？
- [ ] **第二人称叙事**：玩家叙述用"你"？
- [ ] **禁词检查**：是否出现"你做到了 / 完美 / 战略 / 友谊 / 成长 / 突破"？
- [ ] **_IRONY / _BUREAUCRATIC 标注**：合规化用得对吗？
- [ ] **选项克制**：选项 ≤ 4 字 + 不评价 + 不解释？
- [ ] **后果克制**：后果 1-2 行陈述，让玩家自己脑补？
- [ ] **励志检测**：是否出现"努力会得到回报 / 团结就是力量 / 大家一起赢"调子？

任意一条不通过 → 重写。

## 写在最后

这游戏的写作不像写小说。**最锋利的话往往最简单**：

> "她笑了一下，没说话。"
> "工位空了一周。"
> "上一个坐这位置的也是这么想的。"

不要试图"写得好"。**写得真**就够了。这才是反 KPI 的精髓——**做得好不如做得真，做得真不如刚刚好**。
