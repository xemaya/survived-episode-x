# 垂直切片 — NPC 设定（5 人）

> Status: 第 1 版
> Author: Game Designer (Claude)
> Last Updated: 2026-05-04
> Source: design/gdd/npc-relationship-system.md（NPC 算计原则、lifecycle、score）+ game-concept.md（Pillar 2/3/4）

## 设定原则

每个 NPC 必须满足 GDD 的 **NPC 算计原则**：

> "她的行为是为了**她自己**，还是为了玩家？" 如果让玩家觉得"她在帮我"→ 改写。如果让玩家觉得"她在算计 / 她有自己的考虑"→ 通过。

文案不写 "+5 关系"，写 "Lisa 抬起头看了你一眼"。**主语永远是 NPC**。

## 1. Lisa

**基本信息**：女，24 岁，新人 / 同你一批入职 3 个月、同岗位（产品助理）、同部门
**Sprite**：`assets/sprites/npc/faces/` + `positions/` 用 rookie 那套（3 表情：忐忑 / 强笑 / 茫然）
**视觉标签**：
- 头发剪短了（入职第 1 周还是长发，第 2 周突然剪短——"想要一个新的开始"）
- 桌上放着一只玩偶（看起来很小心地摆好，每天位置都不一样）
- 喝奶茶不喝咖啡（你的 mug 满，她的桌上是奶茶杯）

**内在驱动**（她自己的事，不是为你的事）：
- 房贷压力（独居、靠这份工作）
- 大学同班同学已经升 P6，她还在 P4
- 妈妈最近问她"工作稳不稳"
- 真的努力，但她不知道她在努力一个**反向计分系统**

**口头禅 / 行为标签**：
- "诶你看下这个行不行……"（请教时永远先递茶，先试探）
- "我感觉我可能不太适合"（每周说一次，一开始你觉得是谦虚，后来你意识到她是真的觉得）
- 周报永远写到 8 点（她在自己加班，但不申报，怕老板觉得她效率低）
- 微信只发**一句话**，从不发表情包

**本切片关系曲线**（score 起点 +0）：
- Day 1：在茶水间撞见，她让你先接热水。score +0
- Day 2：午休主动问你"要不要拼单"。+5 if 接受
- Day 3：傍晚找你帮她看个 PPT。+5 / -5 / -10 三选
- Day 4：早晨发现她头发剪短了，她说"想换个心情"。flavor only
- Day 5（周五）：偷偷告诉你"HR 找我谈话了"。**关键信息释放**
- Day 6（周六）：微信发"明天来公司加班吗？我自己一个人"。+10 / -5 二选
- Day 7（月末）：早晨工位空着；午饭时她回来了，说"上午去 HR 那签了字"。**结局触发**

**离职 / 留下条件**（day 7 月末考核同时判定）：
- 留下：score ≥ +20 AND 玩家 day 6 选了"陪她加班" AND 玩家 day 3 选了"接过来帮她看"
- **空了**（otherwise）：她说"那我先走了" → 周一早晨工位上玩偶还在，桌面被擦干净；李阿姨在拖地，看见你来了说"这是新规定，每天下班要清桌面"

**未来弧光种子**（month 2+ 如果留下）：
- 她真的来公司加班——你逐渐发现她在准备跳槽面试
- 第 2 个月某周末她"忽然没动静"——后来在朋友圈看到她在另一家公司的工位照
- 那时你会想起 day 6 那个晚上

**❌ 禁忌**：
- 不要让 Lisa 说"谢谢你救了我"——她不会知道是你救的
- 不要让玩家通过帮 Lisa 解锁"协作 +1 AP"或类似 buff
- Lisa 留下不是奖励，**是延长扎心**（她下个月还是会走，你被迫看第二次）

## 2. David（大伟）

**基本信息**：男，28 岁，卷王 / 高你一级（产品经理）、入职 4 年、同部门
**Sprite**：`assets/sprites/npc/faces/` 用 tryhard 那套（3 表情：自信冷笑 / 假装关切 / 急切焦虑）
**视觉标签**：
- 工位上贴满便利贴（自我激励 + KPI 截图）
- 永远拿着保温杯（喝枸杞养生茶，但同时熬夜到凌晨）
- 穿衬衫但不系最上面那颗扣子（"我有松弛感但其实在卷"）

**内在驱动**：
- 上个月被王总监 cue 过一次"潜力一般"，从此每天加班到 11 点
- 老婆刚生孩子，他需要这个职位
- 他真的相信"努力会被看见"，所以他**真的看不起摸鱼的人**——这让他既危险又可怜
- 他不知道他的努力正在让全部门 KPI 阈值集体上涨

**口头禅 / 行为标签**：
- 自称"David"，朋友叫他大伟时会皱眉
- "兄弟，帮个忙吧，五分钟的事。"（永远是五分钟，永远不止五分钟）
- 会议上总抢着第一个发言，发言完看一眼王总监
- 小群里 @你之后 3 分钟没回消息会私聊"在不在？"

**本切片关系曲线**（score 起点 +0）：
- Day 1：早晨在电梯里碰到，主动说"周末过得怎样？"——他的开场白
- Day 2：上午甩给你一个 PPT 模板"帮我改下，5 点要"。-5 if 拒绝 / +0 if 帮 / +5 if 帮且做得好
- Day 3：周三晨会，他**抢了你的方案**说成是自己的。**戏剧高峰一**
- Day 4：午饭叫你"一起吃啊"——他想试探你昨天的反应
- Day 5（周五）：晨会王总监表扬他，他眼神扫过 Lisa 那边
- Day 6（周六）：朋友圈发"周末来加班的都是兄弟"配自拍 + 工位
- Day 7（月末）：你不申报加班的话，他午饭时阴阳"有些人对自己要求不高啊"

**离职 / 留下条件**：
- David **不会在月 1 走**。他是反 KPI 系统的最佳燃料——他自己卷死他自己（month 4-5 燃尽）
- 玩家 score 高低**不影响**他的去留——他根本不 care 你

**未来弧光种子**：
- Month 2-3：他会逐渐从"反派"变成"可怜的同类"
- Month 4：他被王总监批评"潜力已尽"，开始焦虑
- Month 5：他空了。朋友圈最后一条是"开启人生新篇章"

**❌ 禁忌**：
- 不要让 David 自我反省"我也不容易"——他到死都觉得是别人的错
- 不要让玩家"教育"David——他不会听
- 玩家"跟 David 对着干"不该有道德高位——你们俩本质都是在被同一台机器搅

## 3. 王总监（Eric / 真名王大明）

**基本信息**：男，45 岁，部门总监 / 玩家+Lisa+David+Zoe 的直接老板
**Sprite**：`assets/sprites/npc/faces/` 用 boss 那套（3 表情：商务微笑 / 严肃审视 / 假关切）
**视觉标签**：
- 工位是独立办公室（远端，不在工位区）
- 桌上放高尔夫小摆件（暗示中产 aspirations）
- 衬衫一定挽袖子（"我也是干活的"姿态）

**内在驱动**：
- 真名王大明，自封 Eric——"国际化"姿态，老板是华侨海归 CEO
- 部门 KPI 是他的 KPI——他真的不 care 你死活，他 care 数字
- 他自己也被上面的人 PUA，**他是中间层**——他的 PUA 话术是从他老板那里学来的
- 他内心是焦虑的（45 岁面临公司年轻化），但绝不会让你看出来

**口头禅 / 行为标签**：
- 每句话开头"我跟你说啊"或"You know what I mean"
- 称呼员工时混用中英文："小李，你的 deliverable 这周能交吗？"
- 周一晨会必讲"我们是命运共同体"_IRONY
- 给你"赋能" / "对齐" / "拉齐" / "颗粒度" / "抓手" / "闭环"——bullshit bingo 全套

**本切片关系曲线**（score 起点 +0；他的 score 几乎不变，因为他不 care 你）：
- Day 1：午餐时间路过你工位，"小[姓]，加油啊。"——这是他记住你名字的极限
- Day 3（周三晨会）：宣布"本月 KPI 还有 4 天"，"我相信大家的潜力"
- Day 5（周五晨会）：表扬 David——同时眼神扫过 Lisa
- Day 7（月末早晨）：路过你工位"今天月末了，你的 KPI 怎么样？"——他不真的关心，他要的是你的回答让他知道下个月要不要 push 你

**离职 / 留下条件**：
- 王总监**永远不会走**（他是系统的化身）

**❌ 禁忌**：
- 不要让王总监有"家庭温情"刻画（他 45 岁，可能有妻儿，但**永远不在屏幕上提**——他是系统）
- 不要让王总监"识破"玩家——他不在乎你的小聪明
- 不要让玩家"反杀"王总监——这是反 KPI 系统，不是 Boss 战

## 4. Zoe

**基本信息**：女，26 岁，HR 专员 / 入职 1 年、HRBP 团队
**Sprite**：`assets/sprites/npc/faces/` 用 hr 那套（3 表情：职业微笑 / 谨慎 / 假同情）
**视觉标签**：
- 永远穿黑色西装外套（HR 制服）
- 头发盘起来（试图显成熟，但脸上还有学生气）
- 工牌上挂着公司吉祥物钥匙圈（**HR 自己也在被 PUA**）

**内在驱动**：
- 她的本职：执行公司决策，**不是关心员工**
- 但她还年轻，还没完全 numb——所以她说话时会有微弱的犹豫
- 她也想升职到 HRBP 高级，所以她需要"高效执行"
- **她也不知道自己未来会不会被 AI 替代**

**口头禅 / 行为标签**：
- "我们这边的流程是这样的……" _BUREAUCRATIC
- "理解一下"（PUA 时的标准话术）
- "你这边方便的话……"（其实没得选）
- 每句结尾加"哈"——年轻 HR 的口语习惯，让冰冷的话不那么冰冷

**本切片关系曲线**（score 起点 +0）：
- Day 1：午饭时排队碰到，她主动打招呼"诶你刚入职吧"——记住你名字
- Day 4：午饭后路过她工位，看到她在和别的 HR 说"那这周走完流程"——你不知道说的是谁
- Day 5：**Lisa 透露"HR 找我谈话了"——你意识到 Day 4 那句话**
- Day 7（月末）：Lisa 工位空着的时候，Zoe 经过你工位，看你一眼，没说话——她的"专业"姿态强迫她不能流露任何情绪

**离职 / 留下条件**：
- Zoe 不会在 month 1 走
- **未来弧光种子**：随着玩家工龄增加，会出现"高级 HR" 接手玩家的 case——王姐、Mary 等。Zoe 会逐渐被边缘化（年轻 HR 不参与高决策）

**❌ 禁忌**：
- 不要让 Zoe 给玩家"暖心提醒"——她的工作是执行，不是预警
- 不要让玩家"通过 Zoe 套话"打听 Lisa 的情况——HR 不会泄露
- 不要让 Zoe 同情心爆发——她还没到那个阶段

## 5. 李阿姨

**基本信息**：女，52 岁，外包清洁 / 公司清洁阿姨、不属于任何部门
**Sprite**：`assets/sprites/npc/faces/` 用 cleaning_auntie 那套（2 表情：平静 / 关切）
**视觉标签**：
- 蓝色清洁服 + 工牌挂在外面（外包公司的工牌，不是本公司）
- 推着一辆装拖把和清洁剂的小车（车上贴着她孙女的照片）
- 永远穿橡胶手套，但下班前会换成普通鞋

**内在驱动**（她整个人是剧本本身——见 GDD 清洁阿姨 sub-framing）：
- 儿子今年高考，她在想他能不能考上一本
- 来这家公司打扫了 8 年，看过太多人来人去
- 她**完全不懂 KPI 是什么**——她的世界是另一个，但她偶尔会说出最锋利的话
- 她不是为玩家而存在的 NPC——**她有她自己的生活**

**口头禅 / 行为标签**：
- "年轻人，又加班啊"（不带评价，纯陈述）
- "上一个坐这位置的也是这么想的"（**Pillar 3 archive 的人形入口**——她见过太多人）
- 拖地时会哼歌（她哼得不准但很投入）
- 跟玩家说话时**不抬头**（她在干活，但她在听）

**本切片关系曲线**（score 概念存在但**不影响任何机制**——她整个人是 Pillar 2 极致表达，纯叙事，无 game effect）：
- Day 1：傍晚下班路过茶水间，她在拖地，"年轻人，慢点。"
- Day 3：周三早晨，你早到，看到她在擦你的工位，"昨天有人在这吃东西啊"——其实那是 David 中午来你这写过 PPT
- Day 5（周五）：下班时她在收垃圾，你听到她和另一个清洁阿姨低声说"这周又走了仨"——她不是在跟你说，但你听见了
- Day 7（月末）：Lisa 走的那天下午，李阿姨在拖地，**看见你来了，没说话，多拖了一遍 Lisa 的工位附近**

**离职 / 留下条件**：
- 李阿姨在 month 1 内不会走
- **未来弧光种子**：month 6-8 之间她退休——儿子考上大学，她回老家。她离开那天**不会有任何 UI 提醒**，玩家某个早晨发现没人倒水了，自己泡咖啡时才意识到

**❌ 禁忌（GDD 已明确）**：
- 李阿姨**不该影响任何 KPI / AP / 关系数值**——她整个人是剧本
- 不要让李阿姨给"人生哲理"金句——她说的话必须像真实的清洁阿姨说话（朴素、不刻意）
- 不要为李阿姨做特殊 UI、特殊 BGM、特殊 cut——她离开就和她在一样，**安静**

## NPC 同框矩阵（哪些事件会让多个 NPC 同时出现）

| Day | 事件 | 同框 NPC |
|---|---|---|
| Day 1 | 茶水间撞见 | Lisa + 李阿姨（她在拖地） |
| Day 3 | 晨会 | 王总监 + David + Lisa + 你 |
| Day 4 | 午饭后路过 HR 工位 | Zoe（你能听见她和别的 HR 在说） |
| Day 5 | 晨会 | 王总监 + David + Lisa（被 cue 了眼神） + 你 |
| Day 5 下午 | Lisa 透露 HR 找她 | Lisa + 你 |
| Day 7 | 月末早晨 | 王总监路过 + 你 |
| Day 7 中午 | Lisa 回来签字 | Lisa + 你 |
| Day 7 下午 | Lisa 工位空 | 李阿姨多拖一遍 + 你 |

## 设计师自检（写完后自己问）

- [ ] 5 个 NPC 的口头禅是否能让人盲听辨认？（你只看一句对白能判断是谁说的）
- [ ] 5 个 NPC 各自的 motive 是否都"自私"？（不为玩家服务）
- [ ] Lisa 的曲线是否会让玩家在 day 7 那天**胸口一紧**？
- [ ] David 是否让玩家既厌恶又能识别"我身边就有这种人"？
- [ ] 王总监是否真的"系统"——空洞、流程化、无个性？
- [ ] Zoe 是否让玩家觉得"她也是齿轮"——而不是"她是反派"？
- [ ] 李阿姨是否被克制到了"她就在那里干活"的程度？

如果以上 7 条任意一个不通过 → 改写 NPC 设定，再写月剧本。
