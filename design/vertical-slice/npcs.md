# 垂直切片 — NPC 设定（10 人：5 深 + 5 龙套）

> Status: 第 2 版
> Author: Game Designer (Claude)
> Last Updated: 2026-05-05
> Source: design/gdd/npc-relationship-system.md + game-concept.md + `series-structure.md`（52 集 macro）+ `tone-bible.md`（v2 去禁词化）
>
> **v2 changes**：
> - 加 5 龙套 NPC（§6-§10：Vivian / IT 小马 / 老周 / 妈妈 / 林姐）
> - 5 深 NPC 的"未来弧光种子"重对齐到 series macro（Lisa S1-S3 / David S1-S6 / 王总监 S1-S9 / Zoe S1-S12 / 李阿姨 S1-S8）
> - 撤销 `_IRONY` / `_BUREAUCRATIC` 后缀（per tone-bible v2）

## 设定原则

每个 NPC 必须满足 GDD 的 **NPC 算计原则**：

> "她的行为是为了**她自己**，还是为了玩家？" 如果让玩家觉得"她在帮我"→ 改写。如果让玩家觉得"她在算计 / 她有自己的考虑"→ 通过。

文案不写 "+5 关系"，写 "Lisa 抬起头看了你一眼"。**主语永远是 NPC**。

## 1. Lisa

**基本信息**：女，24 岁，新人 / 同你一批入职 3 个月、同岗位（产品助理）、同部门
**Sprite**：`assets/sprites/npc/faces/` + `positions/` 用 rookie 那套（3 表情：忐忑 / 强笑 / 茫然）
**视觉标签**：
- 头发剪短了（入职第 1 周还是长发，第 2 周突然剪短——"想要一个新的开始"）
- 桌上放着一只玩偶（看起来很小心地摆好，每天位置都不一样）
- 喝奶茶不喝咖啡（你的 mug 满，她的桌上是奶茶杯）

**内在驱动**（她自己的事，不是为你的事）：
- 房贷压力（独居、靠这份工作）
- 大学同班同学已经升 P6，她还在 P4
- 妈妈最近问她"工作稳不稳"
- 真的努力，但她不知道她在努力一个**反向计分系统**

**口头禅 / 行为标签**：
- "诶你看下这个行不行……"（请教时永远先递茶，先试探）
- "我感觉我可能不太适合"（每周说一次，一开始你觉得是谦虚，后来你意识到她是真的觉得）
- 周报永远写到 8 点（她在自己加班，但不申报，怕老板觉得她效率低）
- 微信只发**一句话**，从不发表情包

**本切片关系曲线**（score 起点 +0）：
- Day 1：在茶水间撞见，她让你先接热水。score +0
- Day 2：午休主动问你"要不要拼单"。+5 if 接受
- Day 3：傍晚找你帮她看个 PPT。+5 / -5 / -10 三选
- Day 4：早晨发现她头发剪短了，她说"想换个心情"。flavor only
- Day 5（周五）：偷偷告诉你"HR 找我谈话了"。**关键信息释放**
- Day 6（周六）：微信发"明天来公司加班吗？我自己一个人"。+10 / -5 二选
- Day 7（月末）：早晨工位空着；午饭时她回来了，说"上午去 HR 那签了字"。**结局触发**

**离职 / 留下条件**（**S3 finale = E12** 才决定，不是 S1 finale）：
- S1（E1-4）：状态 OK，王总监第一次 PUA"潜力一般"（S1 finale 轻扎）
- S2（E5-8）：状态下滑，KPI 紧张但还在。**S2 finale = 剪短发 + HR 月度面谈（口头警告）**
- S3（E9-12）：HR 真正介入。E9 第二次谈话 / E10 试用期评估 / E11 走/留决定的累积 / **E12 finale = 走/留 climax**
- 留下（路径 A）：累积 score ≥ +20 AND 帮她改 KPI 自评（E10）AND 陪她周末加班（E11）→ 转岗客户成功部林姐处
- 走（路径 B 及其他）：走完离职流程 → S5+ 朋友圈偶尔出现，工位换新人

**未来弧光种子**：
- 路径 A 留下：S5+ 偶尔回办公室办手续；**Endgame Variant C**：E52 春节微信发"新年快乐"
- 路径 B 走：Series 内不再直接出现，但她的工位空 1 周后新人入职——**Series cliffhanger 钩子**

**❌ 禁忌**：
- 不要让 Lisa 说"谢谢你救了我"——她不会知道是你救的
- 不要让玩家通过帮 Lisa 解锁"协作 +1 AP"或类似 buff
- Lisa 留下不是奖励，**是延长扎心**（她下个月还是会走，你被迫看第二次）

## 2. David（大伟）

**基本信息**：男，28 岁，卷王 / 高你一级（产品经理）、入职 4 年、同部门
**Sprite**：`assets/sprites/npc/faces/` 用 tryhard 那套（3 表情：自信冷笑 / 假装关切 / 急切焦虑）
**视觉标签**：
- 工位上贴满便利贴（自我激励 + KPI 截图）
- 永远拿着保温杯（喝枸杞养生茶，但同时熬夜到凌晨）
- 穿衬衫但不系最上面那颗扣子（"我有松弛感但其实在卷"）

**内在驱动**：
- 上个月被王总监 cue 过一次"潜力一般"，从此每天加班到 11 点
- 老婆刚生孩子，他需要这个职位
- 他真的相信"努力会被看见"，所以他**真的看不起摸鱼的人**——这让他既危险又可怜
- 他不知道他的努力正在让全部门 KPI 阈值集体上涨

**口头禅 / 行为标签**：
- 自称"David"，朋友叫他大伟时会皱眉
- "兄弟，帮个忙吧，五分钟的事。"（永远是五分钟，永远不止五分钟）
- 会议上总抢着第一个发言，发言完看一眼王总监
- 小群里 @你之后 3 分钟没回消息会私聊"在不在？"

**本切片关系曲线**（score 起点 +0）：
- Day 1：早晨在电梯里碰到，主动说"周末过得怎样？"——他的开场白
- Day 2：上午甩给你一个 PPT 模板"帮我改下，5 点要"。-5 if 拒绝 / +0 if 帮 / +5 if 帮且做得好
- Day 3：周三晨会，他**抢了你的方案**说成是自己的。**戏剧高峰一**
- Day 4：午饭叫你"一起吃啊"——他想试探你昨天的反应
- Day 5（周五）：晨会王总监表扬他，他眼神扫过 Lisa 那边
- Day 6（周六）：朋友圈发"周末来加班的都是兄弟"配自拍 + 工位
- Day 7（月末）：你不申报加班的话，他午饭时阴阳"有些人对自己要求不高啊"

**离职 / 留下条件**：
- David **S1-S5 全程在**——他是反 KPI 系统的最佳燃料，他自己卷死他自己
- 玩家 score 高低**不影响**他的去留——他根本不 care 你
- **S6 finale = E24 燃尽离职**

**未来弧光种子**（按 series macro）：
- **S1（E1-4）**：抢功 + 假感谢戏码 + 唯一一次试探"潜力一般" 露馅
- **S2-S3（E5-12）**：他变本加厉卷——晨会发言更频繁、加班更晚、在 Lisa 离职那段表现"我是稳定的"
- **S4（E13-16）**：David 燃尽前兆——晨会被王总监打断 / 茶水间摔保温杯 / 朋友圈频率突降
- **S5（E17-20）**：他开始迟到 / 周报敷衍 / 不主动 @所有人
- **S6 finale（E21-24）**：他空了。朋友圈最后一条"开启人生新篇章"。微信群里他的头像变灰

**❌ 禁忌**：
- 不要让 David 自我反省"我也不容易"——他到死都觉得是别人的错
- 不要让玩家"教育"David——他不会听
- 玩家"跟 David 对着干"不该有道德高位——你们俩本质都是在被同一台机器搅

## 3. 王总监（Eric / 真名王大明）

**基本信息**：男，45 岁，部门总监 / 玩家+Lisa+David+Zoe 的直接老板
**Sprite**：`assets/sprites/npc/faces/` 用 boss 那套（3 表情：商务微笑 / 严肃审视 / 假关切）
**视觉标签**：
- 工位是独立办公室（远端，不在工位区）
- 桌上放高尔夫小摆件（暗示中产 aspirations）
- 衬衫一定挽袖子（"我也是干活的"姿态）

**内在驱动**：
- 真名王大明，自封 Eric——"国际化"姿态，老板是华侨海归 CEO
- 部门 KPI 是他的 KPI——他真的不 care 你死活，他 care 数字
- 他自己也被上面的人 PUA，**他是中间层**——他的 PUA 话术是从他老板那里学来的
- 他内心是焦虑的（45 岁面临公司年轻化），但绝不会让你看出来

**口头禅 / 行为标签**：
- 每句话开头"我跟你说啊"或"You know what I mean"
- 称呼员工时混用中英文："小李，你的 deliverable 这周能交吗？"
- 周一晨会必讲"我们是命运共同体"
- 给你"赋能" / "对齐" / "拉齐" / "颗粒度" / "抓手" / "闭环"——bullshit bingo 全套
- 叫笑天名字永远叫不准："小笑啊…陈天啊…差不多差不多"

**本切片关系曲线**（score 起点 +0；他的 score 几乎不变，因为他不 care 你）：
- Day 1：午餐时间路过你工位，"小[姓]，加油啊。"——这是他记住你名字的极限
- Day 3（周三晨会）：宣布"本月 KPI 还有 4 天"，"我相信大家的潜力"
- Day 5（周五晨会）：表扬 David——同时眼神扫过 Lisa
- Day 7（月末早晨）：路过你工位"今天月末了，你的 KPI 怎么样？"——他不真的关心，他要的是你的回答让他知道下个月要不要 push 你

**离职 / 留下条件**：
- 王总监 S1-S8 全程在——他熬过 8 个月
- **S9 finale = E36 王总监被换**——他不是被"打败"，他是被上面老板换掉。他自己也是 puppet

**未来弧光种子**：
- **S1-S3**：标准 PUA 中层——讲反向 KPI 真相 / 表扬 David / cue Lisa
- **S4-S6**：David 燃尽期间他还在 push KPI——他自己也开始焦虑（笑天偶尔看到他自己工位灯到深夜还亮）
- **S7-S8**：他的 KPI 也悬。他开始更频繁找笑天"对齐 deliverable"——这是他在收集情报自救
- **S9 finale（E33-36）**：新总监空降。王总监在 finale 那集"小笑啊…陈天啊…我可能要去其他部门 take 一下新方向"——这是他被换掉的中层话术翻译。他离开那天没说再见

**❌ 禁忌**：
- 不要让王总监有"家庭温情"刻画（他 45 岁，可能有妻儿，但**永远不在屏幕上提**——他是系统）
- 不要让王总监"识破"玩家——他不在乎你的小聪明
- 不要让玩家"反杀"王总监——这是反 KPI 系统，不是 Boss 战
- S9 finale 时不要让王总监"觉醒"——他到走的那天还在讲"我们是命运共同体"

## 4. Zoe

**基本信息**：女，26 岁，HR 专员 / 入职 1 年、HRBP 团队
**Sprite**：`assets/sprites/npc/faces/` 用 hr 那套（3 表情：职业微笑 / 谨慎 / 假同情）
**视觉标签**：
- 永远穿黑色西装外套（HR 制服）
- 头发盘起来（试图显成熟，但脸上还有学生气）
- 工牌上挂着公司吉祥物钥匙圈（**HR 自己也在被 PUA**）

**内在驱动**：
- 她的本职：执行公司决策，**不是关心员工**
- 但她还年轻，还没完全 numb——所以她说话时会有微弱的犹豫
- 她也想升职到 HRBP 高级，所以她需要"高效执行"
- **她也不知道自己未来会不会被 AI 替代**

**口头禅 / 行为标签**：
- "我们这边的流程是这样的……"
- "理解一下"（PUA 时的标准话术）
- "你这边方便的话……"（其实没得选）
- 每句结尾加"哈"——年轻 HR 的口语习惯，让冰冷的话不那么冰冷
- 叫笑天"陈笑天先生"（**笑天内心**：每次她叫这个我就觉得我已经被裁了）
- 工位电脑桌面上偷开小红书《离开大厂的第 100 天，我后悔了吗》

**本切片关系曲线**（score 起点 +0）：
- Day 1：午饭时排队碰到，她主动打招呼"诶你刚入职吧"——记住你名字
- Day 4：午饭后路过她工位，看到她在和别的 HR 说"那这周走完流程"——你不知道说的是谁
- Day 5：**Lisa 透露"HR 找我谈话了"——你意识到 Day 4 那句话**
- Day 7（月末）：Lisa 工位空着的时候，Zoe 经过你工位，看你一眼，没说话——她的"专业"姿态强迫她不能流露任何情绪

**离职 / 留下条件**：
- Zoe **S1-S12 全程在**（HR 是公司最稳的岗位）
- **S6 之后她升级"高级 HR"**——开始接手部门级决策。她出场频率不变但角色升级
- **Endgame**：E51 春节假期前，Zoe 群里发"年终红包"——她的"已为您协调"语气从未变过

**未来弧光种子**：
- **S1**：背景人物。E1 不出场。E2 笑天经过 HR 工位看到她偷刷小红书。E3 偷听到"周五签字"。E4 finale 不出场（她不知道笑天的 KPI 故事）
- **S2-S3**：Lisa 弧光的执行者——E8 找 Lisa 月度面谈 / E10 试用期评估 / E12 finale 走完流程
- **S6+**：升级"高级 HR"。叫笑天还是"陈笑天先生"（年轻 HR 时是礼貌，高级 HR 时是距离）
- **S10-S11**：组织调整传言中她是"内部消息源"——但她从不主动泄密
- **Endgame Variant**（如果玩家对她友好 ≥ 5 次）：她在 E52 春节前一天微信"陈先生新年快乐，注意休息哈"——这是她**唯一**一次去掉"陈笑天先生"全名

**❌ 禁忌**：
- 不要让 Zoe 给玩家"暖心提醒"——她的工作是执行，不是预警
- 不要让玩家"通过 Zoe 套话"打听 Lisa 的情况——HR 不会泄露
- 不要让 Zoe 同情心爆发——她还没到那个阶段

## 5. 李阿姨

**基本信息**：女，52 岁，外包清洁 / 公司清洁阿姨、不属于任何部门
**Sprite**：`assets/sprites/npc/faces/` 用 cleaning_auntie 那套（2 表情：平静 / 关切）
**视觉标签**：
- 蓝色清洁服 + 工牌挂在外面（外包公司的工牌，不是本公司）
- 推着一辆装拖把和清洁剂的小车（车上贴着她孙女的照片）
- 永远穿橡胶手套，但下班前会换成普通鞋

**内在驱动**（她整个人是剧本本身——见 GDD 清洁阿姨 sub-framing）：
- 儿子今年高考，她在想他能不能考上一本
- 来这家公司打扫了 8 年，看过太多人来人去
- 她**完全不懂 KPI 是什么**——她的世界是另一个，但她偶尔会说出最锋利的话
- 她不是为玩家而存在的 NPC——**她有她自己的生活**

**口头禅 / 行为标签**：
- "年轻人，又加班啊"（不带评价，纯陈述）
- "上一个坐这位置的也是这么想的"（**Pillar 3 archive 的人形入口**——她见过太多人）
- 拖地时会哼歌（她哼得不准但很投入）
- 跟玩家说话时**不抬头**（她在干活，但她在听）

**本切片关系曲线**（score 概念存在但**不影响任何机制**——她整个人是 Pillar 2 极致表达，纯叙事，无 game effect）：
- Day 1：傍晚下班路过茶水间，她在拖地，"年轻人，慢点。"
- Day 3：周三早晨，你早到，看到她在擦你的工位，"昨天有人在这吃东西啊"——其实那是 David 中午来你这写过 PPT
- Day 5（周五）：下班时她在收垃圾，你听到她和另一个清洁阿姨低声说"这周又走了仨"——她不是在跟你说，但你听见了
- Day 7（月末）：Lisa 走的那天下午，李阿姨在拖地，**看见你来了，没说话，多拖了一遍 Lisa 的工位附近**

**离职 / 留下条件**：
- 李阿姨 **S1-S7 全程在**——她是公司 background 的稳定声音
- **S8 finale = E32 退休**——儿子考上大学，她回老家
- **她离开那天不会有任何 UI 提醒**——玩家某个早晨发现没人倒水了，自己泡咖啡时才意识到。**这是 Pillar 3 最纯的表达**

**未来弧光种子**（按 series macro）：
- **S1（E1-4）**：标准出场——倒水偶遇 / "这周又走了仨"自言自语
- **S2-S3**：间隔出现，Lisa 离职 finale 路径 B 时**多拖一遍工位**
- **S4-S6**：David 燃尽期间她在背景（她见过太多 David 这种人来去）
- **S7（E25-28）**：她拖把车上贴的"孙女照片"换成了"儿子高考准考证"——铺垫 S8 退休
- **S8 finale（E29-32）**：E29-31 她照常出场。**E32 周一笑天到公司发现没人倒水**——他自己泡咖啡时手停了 3 秒
- **S9+**：换了一个清洁阿姨。新阿姨年轻、不主动说话、戴耳机。**笑天再也没有"上一个坐这位置的也是这么想的"那种瞬间**

**❌ 禁忌（GDD 已明确）**：
- 李阿姨**不该影响任何 KPI / AP / 关系数值**——她整个人是剧本
- 不要让李阿姨给"人生哲理"金句——她说的话必须像真实的清洁阿姨说话（朴素、不刻意）
- 不要为李阿姨做特殊 UI、特殊 BGM、特殊 cut——她离开就和她在一样，**安静**

## 5.5 食堂阿姨（v2.1 追加 — ambient flavor NPC，**不正式注册**）

> Round 1 daily-choices 分身在 #29 引入"食堂阿姨"作为食堂场景的人格化。她不算正式注册的 11 号 NPC，只是一个 **ambient flavor**——跟"打卡机器" / "咖啡机" / "前台水果盘" 同档：背景元素的具象化。
>
> 跟李阿姨形成"两个不在工位的女性"对照——她们是公司"地基的地基"，但她们的世界**从不被 KPI 触及**。

**基本信息**：女，~50 岁，公司食堂阿姨 / 不属于任何部门 / 外包餐饮公司
**出现位置**：仅食堂 daily choice（如 #29 多打半勺）

**视觉锚**：
- 蓝色食堂工服 + 一次性头巾
- 抖三下勺子是行业标准（多 1 下 = 偏爱）
- 不戴口罩（疫情后大家也不戴了）

**口头禅**：
- 不主动说话——只在玩家选择"道谢"时**笑一下不说话**
- 偶尔哼歌（哼"再见再见再见"那种老歌）

**内在驱动**（character note，**绝不在剧本里 expose**）：
- 她不知道笑天的名字。她只记得"每周三都来" / "总点西红柿炒蛋" / "笑得比 David 体面"
- 她有 1 个女儿，开美甲店。她攒钱给女儿换了铺面
- **她的世界比所有人都清楚**——她每天在 11:30 看见所有人都"显得忙"

**剧情功能**：
- 食堂场景的人格化锚点
- "多打半勺" 是公司里**唯一不写 KPI 的偏爱**——见 daily choice #29
- 跟李阿姨平行的 background 镜像：李阿姨在工位区，食堂阿姨在食堂——她们都"在场但不在系统里"

**Series 出场**：S1-Series 内偶尔（约每集 0-1 次，仅在玩家中午去食堂时）

**❌ 禁忌**：
- **不要让食堂阿姨叫笑天名字**——她不知道
- 不要让食堂阿姨"突破" 给笑天人生哲理——她只笑不说话
- 不要给她单独的剧情 event——她只在 daily choice 里出现
- 不要让她有 score 系统——她跟李阿姨一样纯叙事

# 5 龙套 NPC（功能性出场，不需多集弧光）

> 龙套 ≠ 装饰。每个龙套都有 1-2 个**剧情功能**，出现就推动事——不出现也不影响主线。深度比 5 主线 NPC 浅一档，但**口头禅 / 视觉锚 / 内在驱动**仍要立得住。

## 6. 前台 Vivian

**基本信息**：女，28 岁，前台行政 / 入职 6 年（公司元老级前台）/ 不属于任何业务部门
**Sprite**：reception sprite（2 表情：职业微笑 / 八卦时小声）
**视觉标签**：
- 永远画淡妆 + 红色口红（前台必备）
- 工位上有一个**水果盘**——她每天补充
- 永远穿香奈儿风外套（其实是淘宝货）
- 工位电脑屏保：自己旅游照片（马尔代夫，10 年前）

**内在驱动**（她自己的事）：
- 入职 6 年没升过——前台没有晋升路径
- 她比所有人都先知道公司动态（融资 / 高层来访 / 老板心情）
- **她也想跑**——但她的位置是公司"门面"，跑了不好招——她被困
- 老公在隔壁公司工作，他们俩"互相不知道对方真相"

**口头禅 / 行为标签**：
- "嗨～来啦～"（永远拖长音）
- "诶你听说了吗"（接到内部消息时压低声音）
- "今天水果盘是草莓哦～"（融资到位的暗号）
- "今天又是苹果"（融资暂停的暗号）
- 跟笑天说话时眼睛飘向门口（她还在 standby）

**剧情功能**：
- **融资八卦 messenger**——水果盘信号源
- **公司动态先知**——老板今天心情、谁来访、谁请假

**series 出场频率**：scattered。多在周一早晨打卡时 / 老板到访时 / 融资节点

**❌ 禁忌**：
- 不要让 Vivian 跟笑天有 romance subplot——她已婚
- 不要让 Vivian 给笑天"忠诚"或"特殊关照"——她跟所有人都"嗨～"
- 不要让玩家"通过 Vivian 改变 KPI"——她不属于业务

## 7. IT 小马

**基本信息**：男，30 岁，IT 运维 / 入职 4 年（外包技术团队）
**Sprite**：it_guy sprite（2 表情：习惯性疲倦 / 有事时认真）
**视觉标签**：

**内在驱动**：
- 他真的会修——但他**故意不修咖啡机**（咖啡机修好 = 大家不找他 = 他要被裁）
- 他主动维护"我永远在忙"的人设——其实他每天 6 点准时下班

- "您先用别的"（永远没"别的"可用）
- "对不起这个不归我管"（其实归他管）
- 跟笑天说话时眼睛看自己机修包

**剧情功能**：
- **咖啡机 running gag owner**——series 内"已派单 N 周不动"是常驻笑点
- **隐性镜像**——笑天偶尔意识到 IT 小马的"装忙"跟自己的"摸鱼"本质一样

**series 出场频率**：scattered。出现在咖啡机 / 打印机 / 网络故障时

**❌ 禁忌**：
- 不要让 IT 小马"一次性把所有问题修好"——这违背他的人设
- 不要让 IT 小马教训别人"你们不懂技术"——他装得很谦卑
- 不要让笑天"主动帮 IT 小马"——这违背"清醒共谋者"逻辑

**Sprite**：veteran_pm sprite（1 表情：沉默）

- 桌面便利贴："过完今天" —— 跟笑天的"活到周五"是不同时间尺度

**内在驱动**（不要在剧本里 expose，但 character note 必须有）：
- 入职 12 年——他看过 4 任 CEO、6 任部门总监、200+ 同事来去
- 他不主动说话不代表他不知道——他**知道得最多**
- 他每天准点 9:00 到、6:00 走，从不迟到、从不加班、从不请假
- 他的妻子 5 年前去世了（**绝不在剧本里提**）
- **他是笑天的"未来自己"——如果笑天活过 10 年，他就成了老周**

**口头禅 / 行为标签**：
- 几乎不说话。回应人时点头或"嗯"

- 跟笑天说话只在**笑天主动找他**时——他停下手上活，看笑天 0.5 秒，**说 1 句话**

**剧情功能**：
- **笑天的"未来自己"镜像**——series 内笑天每次看老周都是看 10 年后的自己
- **沉默的真相**——老周偶尔说一句话，但那句话每次都对

**series 出场频率**：scattered。E1 在背景出场 1 次，之后每集背景出现 1-2 次。**笑天主动找他 ≤ 3 次/季**

**❌ 禁忌**：
- 不要让老周给笑天"人生哲理"——他只说 1 句话，1 句
- 不要让老周变成 mentor——他不教任何人
- 不要让笑天和老周成为"忘年交"——他们是镜像关系，不是友谊
- 不要在剧本里提他妻子——这是 character note，不是台词

## NPC 同框矩阵（10 NPC 间的 cross-interaction）

> 这个矩阵帮分身 CC session 决定"哪些 NPC 可以放在同一场戏里"。**孤岛 NPC 是 worker thinking——好戏总是 cross-NPC 的**。

### 高频同框组合（series 常出现）

| 场景 | 同框 NPC | 用途 |
|---|---|---|
| 晨会 | 王总监 + David + Lisa（S1-S3）+ 老周（背景）+ 你 | 主线高峰场 |
| 茶水间撞见 | Lisa / David / 李阿姨（拖地）/ IT 小马（修咖啡机） | 偶遇笑点产生器 |

| 周日 8:30 视频 | 妈妈 + 你 | 家庭锚 |
| Lisa A 路径转岗谈判 | 王总监 + 林姐 + Zoe + Lisa + 你 | S3 finale 关键场 |

|---|---|

## 设计师自检（写完后自己问）

**5 深 NPC**：

- [ ] 5 个 NPC 的口头禅是否能让人盲听辨认？（你只看一句对白能判断是谁说的）
- [ ] 5 个 NPC 各自的 motive 是否都"自私"？（不为玩家服务）
- [ ] Lisa 的弧光（S1-S3）是否每集都有递进？不能 1-3 集都"她还好"，4 集突然 HR 谈话
- [ ] David（S1-S6）的"燃尽"是否有铺垫？S4 前兆 / S5 失态 / S6 finale，不是突然空了
- [ ] 王总监（S1-S9）是否真的"系统"——空洞、流程化、无个性？S9 finale 时他**没觉醒**？
- [ ] Zoe（S1-S12）是否让玩家觉得"她也是齿轮"——而不是"她是反派"？她从年轻 HR 升级到高级 HR 的过程是否 visible？
- [ ] 李阿姨（S1-S8）是否被克制到了"她就在那里干活"的程度？S8 退休**没有 UI 提醒**？

**5 龙套 NPC**：

- [ ] Vivian 是否只做"信号源"，不卷入主线？（草莓周 / 苹果周）
- [ ] IT 小马是否每次出现都有"已派单"的笑点 callback？
- [ ] 老周是否每集 ≤ 1 句话？（说多了变 mentor，违背设计）
- [ ] 妈妈是否每周日 8:30 都出现？（缺席 = 主角焦虑信号）
